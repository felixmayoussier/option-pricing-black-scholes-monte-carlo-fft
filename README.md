# Pricing d'options européennes : Black–Scholes, Monte Carlo et FFT de Carr–Madan

Ce projet compare plusieurs méthodes de valorisation d'options européennes sous le modèle de Black–Scholes :

- Formule analytique de Black–Scholes
- Simulation de Monte Carlo
- Méthode de Carr–Madan utilisant la Transformée de Fourier Rapide (FFT)

Le notebook présente les fondements théoriques, l'implémentation des différentes méthodes ainsi qu'une étude comparative de leurs performances numériques et de leur complexité algorithmique.

## Contenu

- `pricing_options.ipynb` : notebook principal contenant les explications, les démonstrations numériques et les résultats.

## Technologies utilisées

- Python
- NumPy
- SciPy
- Matplotlib
- Jupyter Notebook
